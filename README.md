# HAXAI LAB

> AI × DX で、日本のものづくりを軽くする。

三重県の製造業でDX推進を担当しながら、  
中小製造業向けの **AI/DX 実装支援** を個人ブランド **HAXAI LAB** として展開しています。

---

## 🛠 何をやっているか

- **製造業 DX 推進** — 業務改革、AI 活用基盤、社内ナレッジ RAG
- **AI エージェント実装** — Claude Code を使った自律開発
- **データ × 戦略** — 意思決定のための可視化・ML モデル
- **AI ネイティブ組織づくり** — 人を育てるのではなく、AI を配属する
